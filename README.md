# With Love & Yarn by M – Crochet Content & E-Commerce Platform
A full-stack web application built using Flask (Backend) and React (Frontend) that serves as a personal crochet hub for showcasing tutorials, patterns, and handmade products. The platform includes a dynamic admin panel to manage all content, making the system fully scalable and easy to maintain.

## Features

* Add and manage crochet products
* Create and view tutorials
* Upload and share crochet patterns
* Admin panel for full content control
* REST API with Flask
* Minimalist and clean UI

---

## Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Flask
* **Database:** SQLite (Development), PostgreSQL (Production)
* **Storage:** Cloudinary
* **Deployment:** Vercel, Render

## Setup

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

---

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

### Admin Panel

```bash
cd admin
npm install
npm run dev
```
